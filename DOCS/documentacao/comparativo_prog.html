Rápido comparativo entre C#, Java, C++

Este módulo demonstra como que praticamente não existem diferenças no jeito de implementar procedimentos, seja em C#, C++ ou Java.

Última revisão: 13/07/2003

Página do projeto: http://tux.lcc.ufrn.br/~iuri/projeto 

 

Muitos dos procedimentos simples que nós usamos em C# são similares as linguagens das quais ele deriva (C++ e Java), em alguns casos os procedimentos são idênticos.
Olhe os exemplos a seguir:

Declarando variáveis:

C# C++ Java
int x; int x; int x;
int x=3; int x=3; int x=3;

Atribuição:

C# C++ Java
numero = 7; numero = 7; numero = 7;

Loop FOR:

C#:

for (int i = 1; <= 10; i++)
    System.Console.WriteLine("O Número i = {0}", i);

C++:

for (int i = 1; <= 10; i++)
    std::cout << "O Número i = " << i << endl;

Java:

for (int i = 1; <= 10; i++)
    System.out.println("O Número i = " + i);

Loop WHILE

C#:

while ( x < 10)
{
    x++;
    System.Console.Write( x );
}

C++:

while ( x < 10)
{
    x++;
    std::cout << x;
}

Java:

while ( x < 10)
{
    x++;
    System.out.print( x );
}

Estrutura de seleção IF

C#:

if ( n < Max )
{
    total += n;
    n++;
}
else {
    total += n;
    n--;
}

C++:

if ( n < Max )
{
    total += n;
    n++;
}
else {
    total += n;
    n--;
}

Java:

if ( n < Max )
{
    total += n;
    n++;
}
else {
    total += n;
    n--;
}

Como você pode notar praticamente não existem diferenças no jeito de implementar procedimentos.
