Definindo C#.

C# é uma linguagem criada por Anders Heljsberg - mesmo criador do Turbo Pascal e arquiteto do Delphi - para a Microsoft, como uma resposta a crescente popularidade do Java. Sendo assim, C# segue a mesma filosofia do Java e procura ser uma versão melhorada desta linguagem,resgatando algumas características do C++, como estruturas de dados mais simples, criando asssim uma nova linguagem.

A Microsoft submeteu C# e a Common Language Infrastructure ao ECMA (European Computer Manufacturers Association) em 2000. ECMA aprovou o padrão C# em 2001 como ECMA-334. A ISO (International Standards Organization) aprovou este padrão ECMA em 2002.
