Plataforma .Net e C#

Cronograma

 

Nosso trabalho dará uma visão geral sobre a tecnologia .Net e posteriormente mostrará como atua a linguagem C# sobre essa plataforma. Fará ainda paralelos com a linguagem C e com Java. 

Etapas:

  • Plataforma .Net.

    • Uma introdução ao .Net.

    • Utilidade dessa plataforma.

    • Comparativo entre o .Net Framework e o Java Virtual Machine.

     

  • C

    • Introdução ao C# - Um breve tutorial.

    • Recursos novos do C# em relação a C++;

    • Comparativo do C# com a linguagem Java (Recursos presentes no C# que inexistem no Java);

    • Portabilidade do C#;
